Animal Matching

Match-3 Puzzle Game • Unity • C#

Overview

Animal Matching is a classic match-3 puzzle game developed in Unity for both PC and Android. The project focuses on building a complete and responsive match-3 gameplay system from scratch, including tile swapping, match detection, and cascading board resolution.

The core objective of the project was to implement a solid technical foundation for match-3 games, with an emphasis on clean system interactions, extensibility, and satisfying gameplay feedback.

Gameplay

  • Classic match-3 gameplay with adjacent tile swapping
  • Automatic match detection for horizontal and vertical patterns
  • Cascading matches triggered by gravity-based tile refill
  • Special bomb tiles created through larger matches
  • Continuous board updates with responsive input handling

Design Breakdown

The design prioritizes responsiveness and clarity, ensuring that each player action produces immediate and readable feedback. Tile movement, match resolution timing, and cascading effects were carefully tuned to create a satisfying gameplay loop.

Rather than focusing on level-based progression, the project emphasizes systemic design, where interactions between matching, gravity, and special tiles create dynamic outcomes without requiring handcrafted content.

Systems Design

Player Systems

Core Systems

  • Grid-based board system managing tile positions and state
  • Match detection system supporting both horizontal and vertical chains
  • Cascade resolution pipeline using coroutines for sequential updates
  • Tile swapping system with validation and rollback for invalid moves

Power-up System

  • Bomb tiles generated from larger matches
  • Area, row, and column clearing effects
  • Integrated with match resolution system for chain reactions
