Overview

Lab Survival is a wave-based top-down shooter developed in Unity, centered around fast-paced combat and a roguelike upgrade system. Players must survive increasingly difficult waves of enemies while adapting their build through randomized upgrades.

The project focuses on combining moment-to-moment gameplay with long-term decision-making, allowing each run to evolve differently based on player choices and upgrade combinations.

Gameplay

  • Wave-based survival against escalating enemy difficulty
  • Roguelike upgrade system with 3 choices after each wave
  • Build variety through combinable weapon modifiers
  • Fast-paced shooting with continuous enemy pressure
  • Replayable runs with different upgrade paths

Design Breakdown

The core design revolves around creating meaningful player decisions through the upgrade system. After each wave, players are presented with three randomized upgrades, encouraging strategic choices that shape the direction of each run.

Upgrades are designed to interact with each other, allowing for emergent builds such as multi-projectile spread attacks or piercing shots that change how players approach combat. This system adds depth without requiring handcrafted levels or scripted encounters.

Systems Design

Combat System

  • Top-down shooting system with responsive player controls
  • Enemy spawning system with increasing difficulty per wave
  • Collision and damage handling for player and enemies
  • Projectile system supporting multiple modifiers

Upgrade System

  • Roguelike upgrade selection with 3 randomized options per wave
  • Stackable upgrades affecting weapon behavior
  • Examples include spread shot (multi-projectile) and pierce (enemy penetration)
  • System designed for easy expansion with new upgrade types
