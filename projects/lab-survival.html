Lab Survival

2D Roguelike Top-down Shooter • Unity • C#

Overview

A 2D action-platformer prototype inspired by Metroidvania games. Built as a solo project focusing on core gameplay systems.

The game emphasizes exploration, ability-based progression, and interconnected world design that encourages backtracking.

Gameplay

  • Exploration-driven progression with ability gating
  • Fast-paced melee combat with dodge mechanics
  • Puzzle trials that alter boss encounters
  • Non-linear level design with backtracking

Engine

Unity 2022.3.62f2

Language

C#

Role

Solo Developer
Game Designer
Programmer

Design Breakdown

The game is built around an ability-driven progression loop that encourages exploration and backtracking.

Players can either fight bosses directly or complete puzzle trials to weaken them, creating a dual approach to progression.

Systems Design

Player Systems

  • Modular ability system (Dash, Double Jump, Wall Jump, Pulse)
  • Physics-based movement with acceleration and variable jump height
  • Checkpoint system handling respawn and progression

Enemy Design

  • Behavior Tree-based AI system instead of FSM
  • Enemies evaluate conditions dynamically (detect → chase → attack)
  • Reusable modular nodes for scalable enemy design
