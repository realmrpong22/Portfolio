The One After

Turn-based Boss Rush • Unity • C#

Overview

The One After is a turn-based boss fight game developed in 48 hours for a game jam, centered around a single evolving encounter. The player faces one powerful boss, learning its patterns and adapting across repeated attempts.

The design focuses on iterative mastery, where each defeat contributes to a deeper understanding of the boss mechanics, creating a progression system built around player knowledge rather than traditional upgrades.

Gameplay

  • Turn-based combat against a single boss encounter
  • QTE-based dodge system requiring timing and reaction
  • Boss gains new mechanics and weaknesses after each player defeat
  • Pattern recognition and adaptation as core progression
  • Short, repeatable runs designed for rapid iteration

Engine

Unity 2023.2.20f1

Language

C#

Role

Game Designer
Programmer

Design Breakdown

The core concept revolves around transforming failure into progression. Instead of traditional leveling systems, the game evolves by expanding the boss’s behavior after each defeat, encouraging players to learn and adapt rather than grind.

The QTE dodge system adds a real-time layer to the turn-based structure, creating tension between planning and execution. This hybrid approach keeps encounters engaging while maintaining clarity in player decision-making.

Systems Design

Combat System

  • Turn-based combat loop with player actions and boss responses
  • QTE-based dodge mechanic integrated into enemy attacks
  • Clear telegraphing of boss actions to support player learning

Boss Design

  • Single boss encounter with evolving behavior patterns
  • New weaknesses and mechanics introduced after each player defeat
  • Designed to encourage pattern recognition and iterative mastery

UI & Presentation

  • Designed and implemented combat UI for turn-based interactions
  • Clear feedback for player actions and incoming attacks
  • Lightweight interface optimized for rapid game jam development
